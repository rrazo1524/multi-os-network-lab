## 🌐 Network Architecture

All virtual machines communicate through an isolated VirtualBox Host-Only network.

## Lab Architecture

```text
                 [ Windows 11 ]
                       |
          ICMP / TCP / SSH / Nmap
                       |
        -----------------------------------------
        |                                       |
[ Ubuntu Server ] ------------------- [ Kali Linux ]
   SSH Server        Logs + Firewall     Security Testing
```

## Design Principles

- Host-Only isolated network
- No dependency on Internet connectivity
- Simulated attacker (Kali Linux)
- Simulated production server (Ubuntu)
- Windows client for administration and testing
- Safe environment for security experimentation
